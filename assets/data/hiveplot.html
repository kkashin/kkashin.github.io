This is a hive plot of the HIV network data for Stat 221. There are 2 pairs of axes, defined by HIV status. The axes within each pair are actually duplicates. I have chosen to plot the hive plot in this fashion as a start because I wanted to visually separate links between individuals of the same HIV status from links between serodiscordant individuals. Along each axis, individuals are first grouped and color-coded by gender. Then, within gender categories, they are sorted by degree (number of friends). Nodes with higher degree attributes are located closer to the origin of the hive plot. You can mouse over both links and nodes to display information in the legend.
